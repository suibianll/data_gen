消化道出血
消化道出血是指发生在食管、胃、小肠、大肠等消化道部位的出血，临床表现以呕血、黑便或便血为主要特征。

一、病因
消化道出血的常见病因包括消化性溃疡（胃溃疡、十二指肠溃疡）、食管胃底静脉曲张破裂、急性胃黏膜病变、胃癌、结肠癌、肠息肉等。

二、症状
1. 呕血：鲜红色或咖啡色呕血，提示上消化道出血。
2. 黑便：柏油样黑便，提示上消化道或小肠出血。
3. 便血：鲜红色便血，提示下消化道出血。
4. 贫血相关症状：头晕、乏力、心悸、气促等。
5. 神志改变：大量失血时可出现精神淡漠、意识障碍。
6. 生命体征变化：血压下降、心率加快、呼吸急促。

三、辅助检查
1. 血常规：血红蛋白降低，红细胞计数减少，平均红细胞体积（MCV）降低（缺铁性贫血时）。
2. 大便隐血试验：阳性。
3. 胃镜/肠镜：明确出血部位及原因。
4. 腹部CT：了解腹腔脏器情况。

四、治疗
1. 一般处理：卧床休息，禁食或流质饮食，吸氧，建立静脉通路。
2. 补液输血：根据出血量给予晶体液、胶体液或浓缩红细胞输注。
3. 止血药物：奥美拉唑（质子泵抑制剂）静脉滴注，生长抑素或奥曲肽（食管胃底静脉曲张时）。
4. 内镜治疗：胃镜下电凝、注射硬化剂、套扎等。
5. 外科手术：内科治疗无效时考虑。
6. 并发症处理：纠正电解质紊乱、防治感染等。

慢性阻塞性肺疾病（COPD）
慢性阻塞性肺疾病是一种以持续气流受限为特征的慢性呼吸系统疾病，气流受限呈进行性加重，与气道和肺组织对有害气体或有害颗粒的慢性炎症反应增强有关。

一、病因
长期吸烟是最主要的危险因素，其他包括职业粉尘暴露、空气污染、反复呼吸道感染、遗传因素（α1-抗胰蛋白酶缺乏）等。

二、症状
1. 慢性咳嗽：长期、反复咳嗽，晨起明显。
2. 咳痰：白色黏液痰，急性加重时可变为黄绿色脓痰。
3. 呼吸困难：进行性加重，劳力性气促，严重时静息下也有气促。
4. 喘息和胸闷：部分患者有明显喘息。
5. 全身症状：体重下降、食欲减退、外周肌肉无力。

三、辅助检查
1. 肺功能（金标准）：FEV1/FVC < 70%，吸入支气管扩张剂后FEV1 < 80%预计值。
2. 胸部X线/CT：肺气肿征象，肺纹理增多、紊乱，可见肺大疱。
3. 血气分析：低氧血症（PaO2降低），晚期CO2潴留（PaCO2升高）。
4. 血常规：急性加重期白细胞升高，长期低氧可致红细胞增多。

四、治疗
1. 戒烟：最重要的干预措施。
2. 支气管扩张剂：长效β2受体激动剂（沙美特罗、茚达特罗）、长效抗胆碱能药（噻托溴铵）。
3. 吸入糖皮质激素：联合长效支气管扩张剂，用于FEV1 < 60%预计值且反复急性加重者。
4. 祛痰药：盐酸氨溴索、N-乙酰半胱氨酸。
5. 氧疗：长期家庭氧疗（每日≥15小时），适用于PaO2≤55mmHg者。
6. 肺康复：呼吸训练、有氧运动。
7. 急性加重期：短效支气管扩张剂雾化吸入、全身糖皮质激素（强的松30-40mg/d，5-7天）、抗生素（有脓性痰时）。

高血压病
高血压病是以体循环动脉压升高为主要特征的心血管综合征，可分为原发性高血压（占95%以上）和继发性高血压。

一、诊断标准
非同日三次测量收缩压≥140mmHg和/或舒张压≥90mmHg。

二、症状
1. 多数患者无明显症状（"沉默的杀手"）。
2. 头痛、头晕、颈项板紧、疲劳、心悸。
3. 视物模糊（高血压视网膜病变）。
4. 靶器官损害：心脏（左室肥厚、心衰）、脑（脑卒中）、肾脏（蛋白尿、肾衰）、血管（动脉硬化）。
5. 高血压危象：血压急剧升高伴靶器官损害。

三、辅助检查
1. 24小时动态血压监测。
2. 心电图：左室肥厚。
3. 超声心动图：左室肥厚、舒张功能不全。
4. 尿常规+蛋白尿：评估肾损害。
5. 血生化：肾功能、电解质、血脂、血糖。
6. 眼底检查：高血压视网膜病变分级。

四、治疗
1. 生活方式干预：减盐（<6g/日）、减重、戒烟戒酒、适量运动、DASH饮食。
2. 降压药物：
   - 一线药物：ACEI（卡托普利、依那普利）、ARB（缬沙坦、氯沙坦）、CCB（氨氯地平、硝苯地平缓释片）、噻嗪类利尿剂、β受体阻滞剂（美托洛尔）。
   - 联合用药：ARB+CCB（如缬沙坦氨氯地平）、ACEI+利尿剂等。
3. 降压目标：一般患者<140/90mmHg；高风险患者<130/80mmHg。
4. 定期随访：每1-3个月复诊，监测血压及靶器官功能。
